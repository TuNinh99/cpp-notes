# 1. 4 nguyên lý OOP

## 🧾 Content

OOP = Object-Oriented Programming.
4 nguyên lý: 
- Encapsulation: đóng gói dữ liệu + method, dùng access modifier (private, protected, public).
- Abstraction: ẩn chi tiết implementation, chỉ expose interface.
- Inheritance: kế thừa để reuse code.
- Polymorphism: 1 interface nhiều hành vi (overload/override).
