# 3. Abstraction khác encapsulation thế nào?

## 🧾 Content

Encapsulation = che giấu và bảo vệ trạng thái của dữ liệu. dùng access modifier (private, protected)
Abstraction = che giấu logic phức tạp, chỉ expose (phơi ra) API cần thiết. dùng interface/abstract class

Hay hỏi để phân biệt
